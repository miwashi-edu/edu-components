# edu components

Welcome to the edu components documentation. This section provides an overview and usage examples of the React components developed for the Nowle project.

## Getting Started

To use the edu components in your project, ensure you have the package installed. If not, you can add it by running:

```bash
npm install @wacoco/edu-components
```
